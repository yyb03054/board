# 자유게시판

### 계획
자바를 배운지 5년이 지나 아직 미숙하지만 게시판에서 모두 나올 수 있다고 하셨기에 욕심부리지 않고 배우겠습니다.

* 개발 도구
  * intelliJ
  * OpenJDK 11
  * Springboot 2.5.2
  * Gitbash

#### 구현 예정 기능
* 💖💖게시물 생성
* 💖💖게시글 수정
* 💖💖게시글 삭제
* 💖💖게시물 조회
* 💖페이지네이션 기능
* 💖공지사항
* 💖카테고리
* 좋아요 기능(추천 기능)
* 댓글 작성
* 이미지 업로드

### 학습내용

* 회원가입
